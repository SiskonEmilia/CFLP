# CH for CFLP

This is an implementation in C++ of Climbing Hill Algorithm for Capacitated Facility Location Problem. The program compiled with g++ has passed all tests on MacOS.

## Usage

### Compile & Run

```cpp
g++ *.cpp *.hpp -std=c++11
./a.out
```
